# KOREATECH-CGH Dataset

RGBD–complex hologram pairs for ML-based computer-generated holography (ML-CGH). 6,000 pairs (train/val/test) with aligned RGB, depth, amplitude, and phase channels. Generated by SPIN Lab (KOREATECH) using a layer-based hologram.

- Resolutions: 256², 512², 1024², 2048²  
- License: **CC BY-NC 4.0** (non-commercial, attribution required)  
- Hugging Face collection: https://huggingface.co/collections/SPIN-Lab/koreatech-cgh
- Arxiv: https://doi.org/10.48550/arXiv.2512.21040

## Source 3D Models

RGB-D scenes are rendered from 3D meshes in the [Google Scanned Objects](https://research.google/blog/scanned-objects-by-google-research-a-dataset-of-3d-scanned-common-household-items/) dataset.
